Cowin Pinger

Cowin Slot Pinger

Please Read

Using this App, you can set an automatic alert system for your preferred district(s)

The script checks the CoWin API every 1 minute

If the notifications are too frequent, you can always go and delete the scheduled job

NOTE: This app works like a music player. If you kill the app, the notificaitons will stop. Just push the app into the background (i.e. switch to another app or the homescreen) and the notifications will continue
